A free, open-source curriculum built to help anyone learn full-stack web development from scratch.

Whether you're a complete beginner or looking to sharpen your skills, this self-paced program guides you through the fundamentals of web development—one project at a time.

Introduction

This platform is built by a global community of contributors who believe in accessible and effective tech education. It empowers self-learners to become skilled web developers while promoting a culture of collaboration, mentorship, and lifelong learning.

Summary
An open-access curriculum designed for aspiring web developers, providing a guided path through interactive lessons, real-world projects, and community-driven support. Learners can progress at their own pace and gain practical, job-ready skills.

Curriculum

The curriculum offers a well-rounded education in web development, covering both client-side and server-side technologies. It is structured into distinct learning tracks:

Foundations

  • Introduction to core web technologies and development practices.
  • Includes HTML, CSS, JavaScript, Git, and the command line.
  • Prepares learners for advanced specialization in full-stack development.

Full Stack with Python & Django

  • Focuses on full-stack web development using Python and Django.
  • Teaches database management, RESTful APIs, authentication, and deployment.
  • Builds scalable and secure applications with real-world use cases.

Full Stack with JavaScript

  • Centered on modern JavaScript development from front-end to back-end.
  • Uses frameworks and tools such as React, Node.js, Express, and MongoDB.
  • Equips learners with skills to develop responsive, data-driven web apps.

Key Features

This learning platform offers a range of features designed to support and enhance your journey into web development:

  • Open and Free

    All learning resources are completely free to use—no signups, subscriptions, or paywalls. Whether you're just exploring or fully committed, you have unlimited access to the entire curriculum.

  • Collaborative Learning

    Connect with a vibrant community of learners, mentors, and contributors. Share your progress, ask questions, get feedback, and build relationships that support your growth as a developer.

  • Project-Based Curriculum

    Theory is paired with hands-on practice through meaningful, real-world projects. Build everything from simple landing pages to dynamic full-stack applications as you progress through the lessons.

  • Career-Oriented Path

    The curriculum is designed with real job skills in mind—covering everything from version control and testing to frameworks, APIs, and deployment—helping you build a portfolio that stands out to employers.

  • Learn at Your Own Pace

    The curriculum is fully self-paced, allowing you to learn whenever and wherever it suits you. Whether you're studying full-time or squeezing in lessons after work, you're in control of your journey.

  • Modern Tech Stack

    Learn using up-to-date tools and frameworks used by professional developers today. From React and Node.js to Git and APIs, you’ll gain practical experience with the technologies companies actually use.
